Program do obsługi zleceń w drukarni.
Możliwość zalogowania się jako pracownik (drukarz, druk cyfrowy, druk offsetowy, laser, introligatornia) i jako admin
Admin ma możliwość dodawania zleceń dla odpowiednich pracowników, każdy pracownik ma możliwaść zaznaczenia, że zadanie zostało wykonanle i możlwość dodania komentarza, króry będzie isę wyświetlał innym przy tym zleceniu.
Admin może potem wpisać nr faktury i zakończyć zkecenie, albo zaznaczyć, ze było ono podzlecone.
